HTML Structure

HTML5

This template uses the HTML5 Boilerplate framework for a rock solid base to support the template. I recommend watching this screencast by Paul Irish which walks you through the basic setup: http://net.tutsplus.com/tutorials/html-css-techniques/the-official-guide-to-html5-boilerplate/. Please note that not all the functions of the boilerplate have been used. Some redundant aspects have been removed for convince.

Whereas HTML5 brings a lot of new advanced features (geolocation, drag and drop, local storage etc) this template mainly utilizes the new markup. What's the difference? Well instead of writing a div with an id of nav or sidebar, we can just use their HTML5 counterparts instead. It reduces markup and feature-proofs your website.

Main Elements

The template is structured using three main elements. The header, page and footer.

The header and footer are fixed to the top and bottom respectively. Two images are used to fade the page under for a nice effect.

The header contains the logo and the navigation. The logo can be changed by editing the H1 value. The navigation simply links to the anchor of each section. To add a new navigation link, use the section id as the href with an # in front.

The footer contains the copyright information and social buttons. The copyright information can be changed at will and social buttons can be added and removed easily. I've included in the download the full set so you can easily add new icons.

The page consists of 5 sections. One for the home, the about section, portfolio, services and contact form.

The homepage is where you can grab your visitors attention and sell yourself or your mission. In the demo I've color highlighted words in the statement to give emphasis.

The about section is where you can describe yourself and your education experience.

The portfolio section uses thumbnails that are 280 x 180 and uses CSS to style the containers. To use the lightbox simply link to the bigger image, nothing else is required.

The services section is designed to clearly lay out what you offer to potential clients. The icons are from the excellent wefunction collection (link in credits).

To change the contact form to send to your address edit the mail.php and edit the email address at line 48.

CSS

I'm using two CSS files in this theme. Style.css is a html5 reset/setup file. Many browser interpret the default behavior of html elements differently. By using a reset CSS file, we can work round this.

The second CSS file base.css contains all the visual styling for the theme. It is separated into section such as:

/*////   - Headings -   ////*/

.some-code {

}

/*////   - Misc -   ////*/

.some-code {

}

/*////   - Nav -   ////*/

.some-code {

}


Etc, etc.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.
